Como usar este projeto

Usando sua IDE preferida

Clone o repositório:

git clone <SEU_GIT_URL>


Entre no diretório do projeto:

cd <NOME_DO_SEU_PROJETO>


Instale as dependências:

npm i


Inicie o servidor de desenvolvimento:

npm run dev


Requisitos: Node.js e npm instalados (instalação via nvm
)

Editando diretamente no GitHub

Abra o arquivo desejado.

Clique no ícone de lápis (Editar).

Faça suas alterações e faça o commit.

Usando GitHub Codespaces

Na página principal do repositório, clique em Code → aba Codespaces → New codespace.

Edite os arquivos dentro do Codespace e faça commit/push das alterações.
